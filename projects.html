Zero Mile Delivery System

A full-stack parcel tracking system with Spring Boot backend and ReactJS frontend. It supports parcel tracking, delivery uploads, JWT-based login, and role-based access (Admin & Vendor). Hosted on AWS EC2.

  • Backend Tech Stack: Java 21, Spring Boot 3.x, Spring Security, JWT, JPA, H2 DB, Lombok
  • Frontend Tech Stack: ReactJS, Axios, React Router DOM, JWT-Decode
